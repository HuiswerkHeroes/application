# client

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run server
```

### Production start
```
node server.js || npm run start
```

### Start paramaters
Example
```
node server.js --seed-db
```
| Paramaters | Description                                           |
|------------|-------------------------------------------------------|
| --seed-db  | Vult de database met data die de database nodig heeft |
